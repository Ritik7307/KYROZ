# UI Wireframes (Text)

## Admin Panel
- Header: Tenant switcher, billing status, profile.
- Left nav: Dashboard, Users, SOP Library, Analytics, Billing, Settings.
- Main dashboard:
  - KPI cards: MRR, Active Users, AI Queries, SOP Retrieval Success.
  - Charts: Query trends, revenue trend.
  - Tables: Recent SOP uploads, recent team activity.

## Chef Panel
- Header: Current kitchen, shift status, notifications.
- Left nav: AI Assistant, SOP Search, Recipes, Inventory, History, Favorites.
- AI Assistant layout:
  - Chat panel with voice input button.
  - Citation panel with SOP references.
  - Side panel for active recipe steps and timers.
